# 游戏操作指南

## 基础操作

### 移动

| 按键 | 功能 |
|------|------|
| W | 向前移动 |
| A | 向左移动 |
| S | 向后移动 |
| D | 向右移动 |
| Shift | 冲刺（按住） |
| Ctrl | 蹲下（按住） |
| Space | 跳跃 |

### 视角控制

| 操作 | 功能 |
|------|------|
| 鼠标移动 | 瞄准/转向 |
| 鼠标左键 | 射击 |
| 鼠标右键 | 瞄准镜/精确瞄准 |
| 鼠标滚轮 | 切换武器 |

### 武器操作

| 按键 | 功能 |
|------|------|
| 1 | 切换到主武器（突击步枪） |
| 2 | 切换到副武器（霰弹枪） |
| 3 | 切换到狙击步枪 |
| 4 | 切换到火箭筒 |
| 5 | 切换到手枪 |
| R | 装填弹药 |

### 其他操作

| 按键 | 功能 |
|------|------|
| E | 互动/拾取 |
| Tab | 查看地图 |
| Esc | 暂停菜单 |
| F11 | 全屏模式 |

## 游戏模式

- 沉浸式单人剧情体验
- 探索赛博朋克城市

- 无尽波次的敌人攻击

- 团队死斗模式

## 游戏技巧

### 移动技巧

1. **保持移动** - 不要站在原地，移动中更难被命中
2. **利用掩体** - 在建筑物和障碍物后躲避
3. **冲刺穿越开阔地** - 快速通过危险区域
4. **跳跃躲避** - 跳跃可以躲避一些攻击

### 战斗技巧

1. **瞄准头部** - 爆头造成更多伤害
2. **控制射击节奏** - 连射会降低精准度
3. **切换适合距离的武器** - 近距离用霰弹枪，远距离用狙击枪
4. **保留手枪弹药** - 手枪有无限弹药，作为最后手段

### 生存技巧

1. **优先消灭狙击手** - 他们的伤害很高
2. **注意身后** - 敌人会从各个方向出现
3. **及时装填** - 在安全时提前装填
4. **利用爆炸物** - 火箭筒可以消灭一群敌人

## HUD说明

- **左上角**：击杀数和分数
- **左下角**：生命值（红色）和护甲（青色）
- **右下角**：弹药数量和武器槽
- **右上角**：小地图
- **中央**：准星和命中标记
- **顶部中央**：游戏消息

## 故障排除

### 游戏卡顿
- 关闭其他浏览器标签页
- 降低浏览器缩放比例
- 使用Chrome或Firefox获得最佳性能

### 鼠标控制问题
- 点击游戏画面锁定鼠标
- 按Esc解锁鼠标
- 检查鼠标灵敏度设置

### 没有声音
- 检查浏览器是否允许音频
- 点击游戏画面激活音频上下文
- 检查系统音量设置
